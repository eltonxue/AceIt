# AceIt
Web application for helping students practice interviewing. Main features include custom question banks and mock interviews that provide feedback based on voice and tone analytics.

# Technologies Used

Front-end:
- HTML/CSS
- Javascript/jQuery
- Boostrap
- SASS

Back-end:
- Node.js
- ExpressJS
- PostgresQL
- Sequelize

Packages:
- Axios
- Client sessions
- Web sockets
- Nodemon

APIs:
- Web Speech API
- IBM Tone Analyzer API
